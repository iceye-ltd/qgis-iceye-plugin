about=The ICEYE Toolbox regroups a collection of tools for exploitation of ICEYE SAR imagery products (SLC, GRD, QLK, CSI, VID) within QGIS.

    Features: STAC Browser (browse and load the ICEYE AWS Open Dataset), Canvas Screenshot, Layer Export (TIFF/PNG/GIF/MP4), Lens Tool (Normal/Focus/Spectrum/Color/Sub-aperture modes), Crop Tool, Canvas Rotation and SAR Mandala, Height Ruler, IRF Analysis, Batch Processing, Temporal Properties, and Auto Style with SAR-optimised colormaps.

    External dependencies:
    - ffmpeg: required for MP4 video export (Layer Export and Crop Tool video mode)
    - ImageMagick: required for animated GIF export
    - Pillow (Python): optional, used for image processing

    SLC-specific tools (Focus, Spectrum, Color, Sub-aperture, IRF, Canvas Rotation, SAR Mandala, Height Ruler) require ICEYE SLC data with embedded metadata.

    For help installing Python packages in QGIS on Windows, see: https://landscapearchaeology.org/2018/installing-python-packages-in-qgis-3-for-windows/

changelog=Version 0.1.0 (2025-02-25):
    - STAC Browser: browse ICEYE STAC catalog on AWS Open Data and load items as raster layers
    - Canvas Screenshot: export the current map canvas view as PNG or JPEG
    - Layer Export: export raster layers to TIFF, PNG, GIF (ImageMagick), and MP4 (ffmpeg)
    - Lens Tool: interactive magnifying lens with Normal, Focus (1D PGA), 2D Spectrum, Range Power Spectrum, Color, and Sub-aperture modes
    - Crop Tool: draw extent to crop, focus (PGA), create video, or generate RGB spectrum visualisation
    - Canvas Rotation and SAR Mandala: rotate shadows down/layover up/reset north; place and drag georeferenced mandalas
    - Measuring Tools: Height Ruler (SAR-geometry-aware) and IRF Analysis (HPBW, PSLR, ISLR)
    - Batch Processing: collect areas on the map and run Crop, Color, Focus, or Video on each sequentially
    - Temporal Properties: auto-configure temporal ranges for standard and multi-frame ICEYE layers
    - Auto Style and Colormaps: default grayscale styling for SLC layers; Grey Log, Square-Root, Asinh, LogLog, and RGB Teal colormaps
